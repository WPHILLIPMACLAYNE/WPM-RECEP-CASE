# WPM-RECEP-CASE

Repositório oficial do estudo de caso atualizado do projeto **WPM Gestão Interna**.

## Finalidade
Este repositório reúne a versão principal e atualizada do case do projeto, com foco em:
- contexto do problema
- construção da solução
- impacto operacional e humano
- decisão de produto
- resultado final da interface

## Arquivo principal publicado
O deploy público deste repositório utiliza o arquivo `index.html` como entrada principal.

## Observação importante
Existem arquivos HTML adicionais neste repositório por histórico de publicação e upload. O arquivo de referência oficial para o case publicado é o `index.html`.
